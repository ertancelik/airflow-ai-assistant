\# 🤖 airflow-ai-assistant



> The first open-source AI Assistant embedded directly inside the Apache Airflow UI.



Instead of clicking through menus and logs, just chat:



💬 \*"payment\_auth\_pipeline neden fail oldu?"\* → AI fetches the log, analyzes root cause, suggests a fix  

💬 \*"En çok çalışan DAG hangisi?"\* → AI pulls stats and explains  

💬 \*"data\_sync\_pipeline'ı çalıştır"\* → Triggered instantly  

💬 \*"Sistemimin genel durumu nedir?"\* → Full health report of all DAGs  



!\[Airflow AI Assistant Demo](https://raw.githubusercontent.com/ertancelik/airflow-ai-assistant/main/docs/demo.png)



\## ✨ Features



\- 🔍 \*\*Failure Analysis\*\* — Fetches logs, identifies root cause, suggests fixes

\- 📊 \*\*DAG Statistics\*\* — Most run DAGs, longest duration, success/fail rates

\- 🚀 \*\*DAG Management\*\* — Trigger, pause, unpause via natural language

\- 🏥 \*\*System Health\*\* — Overall health report of your Airflow environment

\- 🌐 \*\*Multilingual\*\* — Turkish and English support

\- 🔒 \*\*Privacy First\*\* — Runs 100% locally, no data sent to external APIs
